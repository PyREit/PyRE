PyRE presenta:

Introduction to machine learning for computer vision

Giovedì 28 Febbraio 2019 h19:00

Via Statuto 3 - Reggio Emilia

Sommario

L'incontro sarà un'introduzione ai concetti base di machine learning con un particolare focus sui temi di computer vision. Oltre ad una breve descrizione teorica l'intervento includerà un codelab nel quale si vedrà un esempio base di applicazione di computer vision. I partecipanti potranno portare il proprio laptop per provare in prima persona gli strumenti mostrati.

Agenda
  • Machine Learning & Computer Vision
  • Digit recognition – MNIST data set
  • Artificial Neural Networks
  • Fully connected NN
  • Deep NN
  • Convolutional NN

Livello

Base

Speaker

Michele Vitali è ingegnere per background con un’esperienza professionale nell’ambito della statistica applicata all'innovazione e con la passione per data science, machine learning e artificial intelligence.
